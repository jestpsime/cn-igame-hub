# cn-igame-hub

本仓库用于归档和发布多个独立 HTML 页面。页面内容以信息展示、工具页面或静态演示为主，不针对任何特定网站或域名提供服务。

## 项目简介

cn-igame-hub 是一个静态 HTML 页面集合仓库。所有页面均为独立文件，不依赖后端服务或数据库。仓库主要用途包括：

- 存档各类独立开发的 HTML 页面
- 提供页面内容的静态版本
- 便于页面内容的版本管理和历史追溯

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

**pages/** 目录下每个子文件夹对应一个独立页面，文件夹内包含该页面的完整 HTML 及相关资源。页面之间相互独立，没有跳转或依赖关系。

## 页面归档说明

本仓库中的页面来源多样，可能包括：

- 个人开发的工具或演示页面
- 学习或实验性质的静态页面
- 其他需要存档的 HTML 文档

每个页面均保留原始结构和功能，仅做必要的路径调整以确保在仓库中正常运行。部分页面可能因浏览器兼容性或外部资源失效而无法完全显示，这不属于仓库维护范围。
